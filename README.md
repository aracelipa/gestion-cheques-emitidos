# Sistema de Gestión de Cheques Emitidos

Repositorio del prototipo académico **Sistema de Gestión de Cheques Emitidos**, desarrollado como Trabajo Fin de Grado para la carrera de Análisis de Sistemas Informáticos de la Universidad Autónoma de Encarnación.

## Descripción

El sistema permite registrar, consultar y controlar cheques emitidos por comerciantes de la ciudad de Encarnación. El prototipo incluye gestión de usuarios, control de acceso por roles, registro de cheques, actualización de estados, alertas por correo electrónico, reportes operativos y bitácora de eventos.

El proyecto fue desarrollado como producto mínimo viable y fue probado en entorno local con datos de prueba.

## Tecnologías utilizadas

* PHP
* Laravel
* Blade
* PostgreSQL
* HTML
* CSS
* JavaScript básico
* SMTP externo para envío de correos

## Contenido del repositorio

Este repositorio contiene:

* Código fuente completo del prototipo.
* Archivo `.env.example` con variables de entorno de ejemplo.
* Manual de usuario con instalación, configuración y uso del sistema.
* Documentación técnica básica del sistema.

## Instalación y uso

La instalación completa, configuración del entorno local, ejecución del sistema y uso de los módulos se encuentran detallados en el archivo:

```text
MANUAL_USUARIO.md
```

## Seguridad

El archivo `.env` real no se incluye en el repositorio por motivos de seguridad. Las credenciales de base de datos, correo SMTP y cualquier contraseña deben configurarse localmente.

## Estado del proyecto

Estado: Prototipo académico funcional.

El sistema fue desarrollado para validación en entorno local. No corresponde a una implementación definitiva.
